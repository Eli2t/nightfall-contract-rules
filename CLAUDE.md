# Contract Docs

## What is The Contract?

The Contract is a tabletop RPG where players create "Contractors" — ambitious
individuals who risk their lives in deadly missions ("Contracts") organized by
mysterious entities called Harbingers. Surviving Contractors earn supernatural
"Gifts" — custom-built powers, items, and abilities. The game uses d10 dice
pools, has rotating GMs, and runs on a persistent shared world with an online
platform at thecontractrpg.com.

Key concepts:

- **Contractors** — Player characters. Progress through win brackets (Newbie,
  Novice, Seasoned, etc.) as they complete Contracts.
- **Gifts** — Supernatural powers built from Effects + Enhancements + Drawbacks
  using the Gift Builder on the website.
- **Contracts** — One-session missions. The primary unit of play.
- **Downtime** — Time between Contracts for recovery, training, and side
  activities.
- **Playgroups** — Communities of players that share a setting and house rules.

## What is Nightfall?

Nightfall is a specific playgroup with extensive house rules layered on top of
the base Contract RPG rules. This project documents both the base game's gift
system and Nightfall's modified ruleset.

## Project Structure

### `nightfall-rules/`

The Nightfall playgroup rules reference, split from a single massive Google Doc
into a directory-based structure. Files are numbered to preserve reading order.
See `nightfall-rules/index.md` for the full table of contents.

- `gameplay-basics/` — Overview, dice, contractors, contracts, downtime
- `combat/` — Initiative, actions, reactions, movement, hazards
- `health/` — Injuries, mind, exertion, stress, recovery
- `gifts/` — Gift builder overview, general rules, per-gift clarifications (by
  category), assets & liabilities
- `gifts/clarifications/` — Per-gift errata split by category (masteries,
  awareness, offensive, etc.)
- `stocks/` — Weapons/armor statblocks, battle scars, traumas, stat blocks for
  people/animals/vehicles
- `00-changelog.md` — All house rule changes from the base game

### `gifts/`

Markdown files for every gift effect in the base game (90 effects), generated
from the live site's power system data blob. Organized by category
subdirectories (offensive, defensive, masteries, etc.). Each file lists the
effect's description and all available enhancements and drawbacks with their
descriptions.

Generated by `scripts/generate_gifts.py` from `scripts/power_blob.json`.

### `nightfall-changelog.md`

Root-level copy of the changelog (same as `nightfall-rules/00-changelog.md`).

### `nightfall-diff-report.md`

Full diff report comparing Nightfall rules against the official rulebook.

### `nightfall-changelog-candidates.md`

Checklist of undocumented differences found in the diff, used to triage which
changes should be added to the changelog.

### `scripts/`

Utility scripts for generating and maintaining documentation:

- `generate_gifts.py` — Generates gift markdown files from `power_blob.json`
- `split_rules.py` — Splits the original monolithic rules doc into the directory
  structure
- `cleanup_markdown.py` — Fixes Google Docs markdown export artifacts
- `fix_headings.py` — Normalizes heading levels across all files
- `reformat_weapons.py` — Converts weapon/armor tables to definition-list format
- `replace_links.py` — Replaces thecontractrpg.com rule links with local file
  links

## Working with this project

- The nightfall rules files cross-reference each other with relative markdown
  links. If files are renamed or moved, run a grep for the old path and update
  references.
- The `gifts/` directory reflects the base game as-is from thecontractrpg.com.
  Nightfall house rule changes to specific gifts are documented in
  `nightfall-rules/gifts/clarifications/`, not in the base gift files.
- When updating the changelog, also update the root-level copy
  (`nightfall-changelog.md`) to stay in sync with
  `nightfall-rules/00-changelog.md`.
