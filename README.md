# Contract Docs

Reference documentation for [The Contract RPG](https://www.thecontractrpg.com/),
focused on the Nightfall playgroup's house rules and the base game's gift system.

## Structure

### `nightfall-rules/`

The Nightfall playgroup rules reference, broken down by topic. See
[`nightfall-rules/index.md`](nightfall-rules/index.md) for a full table of contents.

- `gameplay-basics/` — Overview, dice, contractors, contracts, downtime
- `combat/` — Initiative, actions, reactions, movement, hazards
- `health/` — Injuries, mind, exertion, stress, recovery
- `gifts/` — Gift builder, general rules, per-gift clarifications, assets & liabilities
- `stocks/` — Weapons, armor, stat blocks, battle scars, traumas

### `gifts/`

Markdown files for every gift effect in the base game, generated from the live
site's power system data. Organized by category (offensive, defensive, masteries,
etc.), each file lists the effect's description, available enhancements, and
drawbacks.

### `nightfall-changelog.md`

Summary of house rule changes made by the Nightfall playgroup on top of the base
game rules.

### `nightfall-diff-report.md`

Full diff report comparing the Nightfall rules against the official Contract RPG
rulebook, identifying all mechanical differences.

### `scripts/`

Utility scripts used to generate and maintain the documentation.
