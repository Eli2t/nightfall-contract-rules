The file nightfall-changelog.md documents house rules for the "Nightfall" playgroup in The Contract RPG. These are modifications layered on top of the base gift system, not changes to the upstream game data.

**Why:** The gift markdown files in gifts/ reflect the base game system pulled from the live site's power blob. Nightfall's changes (e.g. Signature Item cost caps, Riposte/Counter interaction, Bullet Time difficulty) are playgroup-specific overrides.

**How to apply:** When comparing changelog entries to gift files, treat discrepancies as intentional house rule divergences, not errors to fix. The gifts/ folder is the canonical base game reference; nightfall-changelog.md is the overlay.
